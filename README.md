
# TripGo (Prototype)

تطبيق TripGo للسفر والسياحة - نسخة مبدئية بدون قاعدة بيانات.
يدعم التبديل بين العربية والإنجليزية.

## الألوان
- الأخضر (اللون الأساسي)
- الأصفر (للتمييز)
- الأسود (للنصوص والعناصر الثانوية)

## التشغيل
```
flutter pub get
flutter run
```
أو لبناء APK:
```
flutter build apk --release
```
